# 会话交接

## 新会话开始
1. 读取 Core/记忆/user_scope.md（用户偏好）
2. 读取 Core/记忆/project_scope.md（项目状态）
3. 读取 Core/记忆/session-handoff.md（上次会话摘要，如存在）
4. 告知用户已加载记忆，并恢复上下文

## 会话结束
用户输入 /结束会话 时，调用 `Skill{name="session-handoff"}`，执行：
1. 生成会话摘要（做了什么、关键决策）
2. 记录未完成的任务
3. 记录重要上下文（供下次恢复）
4. 写入 Core/记忆/session-handoff.md

## 会话恢复原则
- 新会话自动恢复上次的未完成工作
- 如需重新开始，用户可输入 /重置
